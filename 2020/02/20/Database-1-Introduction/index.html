Database 1. Introduction

Posted on 2020-02-20 Edited on 2020-03-05 In Database Valine:

引言

本文将介绍数据库的相关基本信息。

专业术语

Data

facts that are recorded and can be accessed
指已记录或者可获取的事实。

  • Data formats – text, numbers, figures, graphics, images, audio/video recordings and more

  • Data is recorded and kept because it is considered to be of use to an intended user

Information

refers to the data that is accessed by a user for some particular purpose

  • Typically, getting the needed information from a collection of data requires performing an activity, such as searching through, processing, or manipulating the data in some form or fashion

为了区分数据和信息，让我们来看一个例子。比如一个电话本是一组收集到的数据——如个人，银行等的电话号码。在特定情况下，这些电话号码就会成为信息。例如，一个拥有电话本的人饿了，决定给披萨店打电话。此时披萨店的电话对他来说就变成了信息。

Metadata

data that describes the structure and the properties of the data

  • Metadata is essential for the proper understanding and use of the data

Data without metadata - example

Database

structured collection of related data stored on a computer medium
指在计算媒介上按结构存储的相关数据。数据库的目的是将数据组织起来，以便用户直接从数据中获得信息。数据库的结构由数据库元数据解释。

  • Organizes the data in a way that facilitates efficient access to the information captured in the data

Database metadata

represents the structure of the database
关于数据的数据，比如表名、列名，数据类型等

  • Database content that is not the data itself (data about the data)
  • Contains:
    • Names of data structures
    • Data types
    • Data descriptions
    • Other information describing the characteristics of the data

Database management system (DBMS)

software used for:

  • Creation of databases

  • Insertion, storage, retrieval, update, and deletion of the data in the database

  • Maintenance of databases

数据库和DBMS的关系

这两者的关系就像是演示软件（DBMS）和演示（Database）之间的关系。演示软件是用来创建演示、插入演示内容、执行演示以及改变或删除演示内容的。

Database system

数据库系统的目的是在用户和数据库信息之间保证高效的交互。其三个主要组成部分是数据库、DBMS和前端应用程序。

  • Database是核心
  • 而数据库的所有交互操作是通过DBMS来完成
  • 前端应用程序是为用户和DBMS之间更易于交互而提供的一种机制

数据库系统开发步骤

  • Conceptual database model（需求可视化） – a visualization of requirements by using a conceptual data modeling technique (such as entity-relationship [ER] modeling)
  • Logical database modeling （数据库建模Database modeling） - creation of the database model that is implementable by the DBMS software

Logical database modeling follows conceptual database modeling

数据库新版本

The new version of the database follows the same development steps as the initial version
